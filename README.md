# Digitron — Simple WPF Calculator

A minimal WPF desktop application built with **C# and .NET 6**, simulating a basic calculator for performing arithmetic operations.

## 🧮 Features
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷) with division-by-zero check
- Clear (⟲) button to reset inputs
- Simple, modern UI with rounded buttons and clean layout

## 🛠️ Tech Stack
- **C# / WPF (.NET 6)**
- **XAML** for UI
- **MVVM-inspired** structure (simple code-behind logic)

## 🚀 How to Run
1. Open `DigitronWPF.csproj` in **Visual Studio 2022+**
2. Press **F5** or choose **Debug → Start Debugging**
3. Enter two numbers, select an operation, and view the result.
